Técnico em Desenvolvimento de Sistemas

O curso Técnico em Desenvolvimento de Sistemas do Colégio Barbosa Ferraz forma profissionais capacitados para atuar no mercado de tecnologia da informação, uma das áreas que mais cresce no Brasil e no mundo.

Nossa grade curricular abrange linguagens de programação modernas como Python, JavaScript, Java e C#, além de tecnologias web, desenvolvimento mobile, banco de dados e metodologias ágeis de desenvolvimento.

Os estudantes têm acesso a laboratórios equipados com computadores de última geração e softwares profissionais, proporcionando uma experiência prática e alinhada com as demandas do mercado.

Durante o curso, os alunos desenvolvem projetos reais, participam de hackathons e têm a oportunidade de realizar estágios em empresas parceiras, garantindo uma formação completa e prática.

Nossos egressos estão preparados para trabalhar como desenvolvedores de software, analistas de sistemas, programadores web e mobile, além de terem uma base sólida para empreender na área de tecnologia.
